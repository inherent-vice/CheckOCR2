## 2025-02-12 - Reusable ToolTip implementation
**Learning:** Reusable ToolTip class is effective for icon-only buttons in Tkinter applications where `bbox` is unavailable, improving accessibility by explaining ambiguous UI elements like '📂'.
**Action:** Use this `ToolTip` component for any other ambiguous icon-only UI elements across the Tkinter interface to ensure users understand the action.
