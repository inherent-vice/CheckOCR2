# 네트워크 폴더 지원 개선

## 📡 네트워크 경로 처리

### UNC 경로 자동 정규화
- 단일 백슬래시 경로 (`\server\share`) → 이중 백슬래시 (`\\server\share`)
- 슬래시 경로 (`//server/share`) → 백슬래시 (`\\server\share`)
- 경로 정규화 자동 감지 및 변환

### 네트워크 접근 검증
- 서버 접근 가능 여부 자동 확인
- 하위 디렉토리 생성 권한 검사
- 연결 실패 시 사용자 친화적 오류 메시지

### 폴더 열기 개선
- Windows 탐색기 통합 (`os.startfile` + `explorer.exe` 백업)
- 네트워크 상태 확인 및 안내 메시지
- 권한 오류 시 상세 가이드 제공

### 지원되는 네트워크 형식
- Windows UNC: `\\server\share\path`
- IP 주소: `\\192.168.1.100\share\path`
- 도메인 서버: `\\domain-server\share\path`

### 문제 해결 가이드
1. **연결 오류**: 네트워크 연결 및 VPN 상태 확인
2. **권한 오류**: 공유 폴더 쓰기 권한 확인
3. **경로 오류**: UNC 경로 형식 확인 (`\\server\share`)
